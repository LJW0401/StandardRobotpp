# 云台介绍
## 硬件规范
### CAN线
- **yaw-pitch直连云台**

  若使用 **GM6020** 作为yaw,pitch轴驱动电机，统一接入`CAN2`，并使用 `0x1FF` 标识符作为底盘数据包的标识符

- **yaw直连-pitch丝杆**
  - 同yaw-pitch直连云台
  
- **yaw-pitch丝杆**
  - 同yaw-pitch直连云台

### 电机
统一以逆时针旋转为正方向(发送正电流时的电机输出轴旋转方向)

### 电机ID
- yaw轴：1
- pitch：2
