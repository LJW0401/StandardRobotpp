### 遥控器使用
- **平衡底盘**
  - 左拨杆：未使用
  - 右拨杆：底盘模式切换。
    - 上档：正常控制，跟随云台
    - 中档：正常控制，自由底盘
    - 下档：底盘无力，紧急停止
  - 左摇杆：未使用
  - 右摇杆：控制x和y方向上的平移速度
  - 左滚轮：小陀螺旋转速度

- **全向轮、麦轮底盘**
  - 左拨杆：未使用
  - 右拨杆：底盘模式切换。
    - 上档： 
    - 中档：
    - 下档：
  - 左摇杆：未使用
  - 右摇杆：控制x和y方向上的平移速度
  - 左滚轮：小陀螺旋转速度
